# Vaultify

**Система адресного хранения вещей**

Telegram-бот для управления предметами и боксами с инвентарём.

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/menu` | Главное меню с кнопками навигации |
| `/add <название>` | Добавить предмет (выбор бокса через меню) |
| `/find <название>` | Поиск предметов по названию и комментарию |
| `/box` | Интерактивный список боксов (inline-кнопки) |
| `/list` | Все предметы по боксам |
| `/newbox <название>` | Создать новый бокс |
| `/cancel` | Отменить текущее действие |

## 🚀 Быстрый старт

### Установка

1. **Создайте директорию и клонируйте проект:**
```bash
sudo mkdir -p /opt/vaultify
cd /opt/vaultify
sudo git clone https://github.com/MisterCron/vaultify.git .
```

2. **Скачайте файл `.env`:**
```bash
sudo curl -sSL https://raw.githubusercontent.com/MisterCron/vaultify/main/.env.example -o .env
```

3. **Настройте файл `.env`:**
```bash
sudo nano .env
```

Отредактируйте параметры:
```bash
BOT_TOKEN=1234567890:AABBccDDeeFFggHHiiJJkkLLmmNNooP  # Токен бота, полученный от @BotFather
BOT_DEV_ID=123456789                                  # Ваш Telegram ID
ALLOWED_USERS=123456789,987654321                     # Список пользователей (опционально)
```

> 💡 Узнать свой Telegram ID можно через бота [@userinfobot](https://t.me/userinfobot)

4. **Запустите контейнеры:**
```bash
sudo docker compose up -d --build && sudo docker compose logs -f -t
```

### Обновление

```bash
cd /opt/vaultify
sudo docker compose down
sudo git pull
sudo docker compose up -d --build
```

## 🔐 Настройка

Файл `.env` содержит следующие параметры:

| Параметр | Описание |
|----------|----------|
| `BOT_TOKEN` | Токен бота, полученный от @BotFather |
| `BOT_DEV_ID` | Telegram ID главного разработчика |
| `ALLOWED_USERS` | Список пользователей через запятую (опционально) |

Если `ALLOWED_USERS` пустой — доступ разрешён всем. Администратор (`BOT_DEV_ID`) всегда имеет доступ.

## 🎯 Функционал

### Главное меню
- Навигация через inline-кнопки
- Разделы: Боксы, Список предметов, Найти предмет
- Кнопка «📋 Главное меню» для возврата из любого раздела

### Управление боксами
- Создание/переименование/удаление боксов
- Просмотр списка предметов в боксе
- Счётчик предметов в названии бокса
- Подтверждения действий с автоудалением (3 сек)

### Управление предметами
- Добавление предметов в бокс
- Переименование предметов
- Добавление/удаление комментариев
- Информация о создателе и дате добавления
- Подтверждения действий с автоудалением (3 сек)

### Поиск
- Поиск по названию предмета
- Поиск по комментарию
- Регистронезависимый поиск
- Поддержка кириллицы

## ⚡ Оптимизации

- **Кэширование БД** — боксы и предметы кэшируются для ускорения доступа
- **Фоновое удаление сообщений** — не блокирует работу бота
- **Поиск через SQL** — быстрая фильтрация предметов
- **Модульная архитектура** — разделение на логические модули

## 📝 Информационные сообщения

Все подтверждения действий работают по единому шаблону:

1. Сначала появляется меню с кнопками
2. Снизу — сообщение с подтверждением
3. Через 3 секунды подтверждение исчезает

| Действие | Сообщение |
|----------|-----------|
| Создание бокса | ✅ Бокс "название" создан! |
| Добавление предмета | ✅ Предмет "название" добавлен! |
| Переименование бокса | ✅ Бокс переименован! |
| Переименование предмета | ✅ Предмет переименован! |
| Сохранение комментария | ✅ Комментарий сохранён! |
| Удаление комментария | ✅ Комментарий удалён! |
| Удаление предмета | 🗑️ Предмет удалён |
| Удаление бокса | 🗑️ Бокс удалён |
